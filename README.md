## Acerca de

**DOSefi** es una página de música lofi diseñada para acompañarte durante tus sesiones de estudio o trabajo. Con un estilo retro, cozy y relajante, reproduce beats calmados que te ayudan a mantener el enfoque.

### Características

- 🎵 Reproducción de música lofi en bucle
- 🎨 Diseño retro con efectos visuales (scanlines, vignette)
- 📺 Video de YouTube como fondo de audio (invisible)
- 🖥️ Controles simples: play/pause y fullscreen
- 🌙 Perfecto para estudiar, trabajar o relajarse

## Cómo usar

1. Abre `index.html` en un navegador web (usando un servidor local)
2. Presiona el botón de play para comenzar a reproducir
3. Usa el botón de fullscreen para una experiencia inmersiva

## Tech Stack

- HTML5
- CSS3 (efectos retro)
- YouTube IFrame API
